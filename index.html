Welcome

This project demonstrates a simple Blockchain Based Certificate Verification System.


Admin can register institutes.

Approved institutes can issue certificates.

Certificates are stored with SHA-256 hash in a blockchain simulation.

Anyone can verify a certificate using its Certificate ID.

Project Workflow

Step Description

2 Admin Approves Institute
3 Institute Issues Certificate
4 SHA-256 Hash Generated
